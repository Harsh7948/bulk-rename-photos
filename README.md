# Bulk Rename Photos

A lightweight web application for renaming multiple image files quickly and efficiently. Designed for photographers, content creators, and businesses handling large numbers of images.

## Features

- Bulk photo renaming
- Fast and efficient processing
- User-friendly interface
- Batch file management
- Easy-to-use workflow
- Built with HTML, CSS, and JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

## Use Cases

- Jewellery photography
- Product image organization
- E-commerce image management
- Bulk image processing
- Content creation workflows

## Benefits

- Save time when managing large numbers of photos
- Organize files efficiently
- Improve productivity
- Simplify file naming tasks
